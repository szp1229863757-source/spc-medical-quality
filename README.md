# 基于增强型统计过程控制的医疗质量分析平台

**Enhanced SPC Medical Quality Monitoring Platform**

> 将制造业的统计过程控制（SPC）方法引入医疗质量管理，用算法自动识别医院绩效异常，把误报率降低 **81%**，让质量监管从"凭经验"变成"看数据"。

---

## 这个项目解决了什么问题

美国联邦政府每年根据 30 天再入院率等指标对全美医院进行经济处罚（HRRP 计划），但传统的质量监控方法存在严重缺陷：**误报太多**。传统 P 控制图在医疗数据上会产生 1,039 次误报——质控人员被大量虚假警报淹没，真正的异常反而被忽略。

本项目提出了一种 **Laney P' + I-MR 混合控制图模型**，结合 Wilson Score 置信区间修正，将误报从 **1,039 次降至 198 次（减少 81%）**，同时构建了医院绩效自动分级系统，实现从原始数据到交互式 HTML 报告的全流程自动化。

## 方法论：为什么传统的 P 控制图不适用于医疗数据

```
传统 P 控制图（用于制造业）:
  假设: 每个产品独立同分布
  问题: 医疗数据存在"过度离散"（overdispersion）
        → 同一家医院的患者不是"独立零件"，他们共享同一批医生/设备/流程
        → 传统 P 图会把这种正常波动误判为"异常"，产生大量假警报

Laney P' 修正:
  1. 先计算传统 P 图的 σ（理论波动）
  2. 再计算 z-score 的移动极差（实际波动）
  3. 两者比例 = σ_z（过度离散因子）
  4. 用 σ_z 修正控制限 → 区分"正常波动"和"真正异常"

Wilson Score 修正（小样本）:
  当某医院某月出院人数 < 10 时，传统 ±3σ 近似失效
  → 改用 Wilson Score 二项比例置信区间，统计推断更稳健
```

## 核心成果：误报率降低 81%

| 对比维度 | 传统 P 控制图 | 增强型 Laney P' 控制图 |
|----------|-------------|----------------------|

| 适用场景 | 制造业均匀产品 | **医疗数据（天然过度离散）** |

## 绩效分级体系

系统自动为每家医院计算综合得分并分级：

| 等级 | 标准 | 建议措施 |
|------|------|---------|
| **Elite** | Ppk ≥ 1.33，Z-score ≥ 3.0 | 标杆医院，推广经验 |
| **Good** | Ppk ≥ 1.0，Z-score ≥ 2.0 | 正常监控 |
| **Warning** | Ppk ≥ 0.67，Z-score ≥ 1.0 | 重点关注，月度审查 |
| **Critical** | Ppk < 0.67 或 Z-score < 1.0 | 立即介入，启动整改 |

## 数据来源

美国 CMS（医疗保险与医疗补助服务中心）公开的 **HRRP（Hospital Readmissions Reduction Program）** 数据集，包含全美 3,000+ 家医院的 6 项再入院质量指标（急性心梗、心力衰竭、肺炎、COPD、CABG、髋/膝关节置换）。
